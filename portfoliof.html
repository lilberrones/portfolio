If you're on this page, congrats to me; this page actually works! Now allow me to explain what I have learned in this class:
Before this class, I never knew about WebStorm. At all. After using it all semester, I realized how great of a developer tool it is. It allows users to create projects, connect to your GitHub, push/pull your data to/from repositories on GitHub, make deployment servers, start an SHH session, and that's just the beginning of it. WebStorm is a versatile tool and is something I'll like to keep practicing on.
GitHub is like what the name implies, your central hub for your git repositories and/or projects. You create an account and you can begin creating repositories here. A repository is basically your main project that will store the file you push up to it, and download the most recent file by pulling it down via WebStorm.
